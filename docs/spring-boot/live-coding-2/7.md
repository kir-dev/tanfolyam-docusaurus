## 7. Fejezet - Logolás

A [Node.js-es tanfolyam végén](https://tanfolyam.kir-dev.hu/docs/node-js/finishing-the-api/logger-middleware) szuper hosszúra nyúlt a **logolás hozzáadása a projekthez**, de Spring Boot-ben **ennél rövidebb megoldást nem is lehetne elképzelni**:

Nincs más teendőnk, mint az `application.properites` fájlba (`resources` mappán belül) **hozzáadni ezt az egy sort**, és működik is a logolás🥳

```kotlin
logging.level.web=debug
```
