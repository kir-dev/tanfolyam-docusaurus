# Általános Tudnivalók

## Tanfolyamaink felépítése

Tanfolyamaink egyaránt tartalmaznak elméleti és gyakorlati részeket a következő felépítésben:

- **Webes alapok** valamint **Git és GitHub alapok** tanfolyamaink 1-1 alkalmasak, ahol a Webes alapokon megírt demo program segítségével mutatjuk be a Git használatát, így a két tanfolyam szorosan összekapcsolódik.
- **Node.js**, **SpringBoot**, **React** tanfolyamaink 3-3 alkalmasak, ahol az első alkalom elméleti, előadás jellegű, a második és harmadik pedig gyakorlati, közös kódolós kurzus, ahol gyakorlati példán keresztül bemutatjuk nektek a modern webfelesztés alapjait.

## Amikre szükséged lesz

- Valamilyen fejlesztőkörnyezetre lesz szükséged, hogy velünk közösen tudj kódolni. Ez lehet **Visual Studio Code**, **WebStorm** vagy bármilyen más, általad preferált IDE.
- Tanfolyamainkhoz kelleni fog egy **GitHub fiók** is, hogy a demo programokat el tudjad érni. Amennyiben szükséged van segítségre, bármelyik tanfolyam végén szívesen segítünk regisztrálni.
